# Project #3 - StudySpot: A Map-based Cafe Finder
[CS 5610: Web Development by John Alexis Guerra Gómez](https://johnguerra.co/classes/webDevelopment_online_spring_2026/)

# Project Objective
The goal of this project was to get comfortable creating a full stack application using HTML, CSS, Bootstrap, JavaScript, MongoDB, Express, and React. It was a good opportunity to learn how to piece together all of the different moving parts and add React to our project skills.

The goal of actual application is to serve as a way for students in particular to find study spots around them. They can add study spots that they find or look for ones around them that fit their needs. A neat feature is that users can alert other users how busy a study spot in real-time.

# How to Build
1. `git clone https://github.com/brooklynschmidt/project3-StudySpot`
2. `cd project3-StudySpot`
3. `npm install`
4. `docker run --name mongodb -p 27017:27017 -d mongodb/mongodb-community-server:latest` Due to this step, you may need to change the MongoDB string that is used to connect to the DB.
5. `npm start`
6. In a different terminal, run npm run dev
7. Open up the local host page

# Instructions on How To Use The Application
1. Press Sign Up (or Login if you have an account)
2. Enter in the information for your account
3. On the left side, there is a map. Look around for colored dots to see any spots in your area
4. If you want to add a spot, click the green "+" button in the top right.
5. Fill out the information as you see fit.
6. If you wish to adjust the real-time busy state, click on the Spot card, and click any of the three "Not Crowded, Moderate, Crowded" icons.
7. If you wish to view information about a spot, click on the Spot card.
8. To search for a specific spot, type in the search bar. You can search by address or name.
9. To favorite a spot, click on the outlined heart icon on the Spot card.
10. To view your favorites list, click the outlined heart in the top right.
11. To filter for spots that fit your needs, select the 3 horizontal bars in the top right. Fill out the information and click Apply Changes.
12. To adjust your profile, click on your profile icon in the top right.
13. Click Edit Profile and add any information you want and adjust your usernameas well.
14. Optionally, you can delete your account if you would like.
15. Click "Log Out" to log out.
